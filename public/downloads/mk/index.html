 About Studio Fojar   

Studio Fojar was made up of several guys who at one point had enough free time on their hands to try creating computer games. Differing design philosophies and real-world commitments slowly killed off the project before it could release more than it's first game, the macintosh-only fighting game, Mutton Kombat. Nearly all the development was done by Miles Johnson, of Isometric fame. A complete game engine was created. If you are interested in learning more about the FCS game engine, email miles@spaceninja.com.
